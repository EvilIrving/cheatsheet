常用场景

  | 场景          | 推荐                         |
  |-------------|----------------------------|
  | 快速临时分享文件    | python -m http.server      |
  | 开发 API/前端项目 | Flask 或 FastAPI            |
  | Django 项目开发 | python manage.py runserver |
  | 生产环境        | Nginx + uWSGI/Gunicorn     |

  最简单的方式就是 python3 -m http.server，无需任何依赖。
